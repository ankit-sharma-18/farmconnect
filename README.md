# 🌾 FarmConnect - Local Food Marketplace

A Django-based platform connecting local farmers directly with conscious consumers.

## 🚀 Features

- 👨‍🌾 Farmer profiles with product catalogs
- 🗺️ Interactive maps with delivery zones
- 🤖 AI-powered recommendations
- 💬 Direct messaging system
- 📦 Order management
- ⭐ Review and rating system

## 📦 Installation
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/FarmConnect.git
cd FarmConnect

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Navigate to Django project
cd farmconnect

# Run migrations
python manage.py migrate

# Create superuser
python manage.py createsuperuser

# Run server
python manage.py runserver
```

## 🛠️ Tech Stack

- Django 4.2.7
- Bootstrap 5
- Leaflet.js
- AI/ML Recommendations
